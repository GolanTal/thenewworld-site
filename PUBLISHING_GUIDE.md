# How to publish this site on GitHub Pages

### Option 1 — Project site (recommended)
- Repo name: anything (e.g., `thenewworld-site`).
- Enable Pages from **Settings → Pages** (Deploy from a branch → `main` / root).
- URL: `https://USERNAME.github.io/REPO/`

### Option 2 — User site
- Repo name **must** be `USERNAME.github.io` (e.g., `golantal.github.io`).
- Pages auto-serves the root at `https://USERNAME.github.io/`
- Only one user site per account.
